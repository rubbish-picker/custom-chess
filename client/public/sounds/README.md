# 音效文件说明

本项目需要两个音效文件：

1. **move.mp3** - 落子音效（棋子移动时播放）
2. **check.mp3** - 将军音效（将军时播放）

## 获取音效的方法

### 方法 1: 使用免费音效网站

可以从以下网站下载免费音效：

- https://freesound.org/
- https://pixabay.com/sound-effects/
- https://mixkit.co/free-sound-effects/

搜索关键词：

- move.mp3: "chess move", "click", "tap"
- check.mp3: "chess check", "alert", "notification"

### 方法 2: 临时使用静音文件

如果暂时不需要音效，可以创建空的音频文件，程序会自动处理播放失败。

### 方法 3: 使用在线音效生成器

- https://sfxr.me/ - 复古游戏音效生成器
- https://www.bfxr.net/ - 音效生成工具

## 文件要求

- 格式：MP3
- 大小：建议小于 100KB
- 时长：建议 0.5-2 秒

将下载的音效文件重命名为 `move.mp3` 和 `check.mp3`，放置在此目录即可。
